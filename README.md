# Restart-PC

Готовый сайт Restart-PC с витриной компьютеров, услугами, регистрацией, личным кабинетом клиента и панелью администратора.

## Запуск на Windows

1. Распакуйте архив.
2. Откройте папку `restart-pc-purple-photo`.
3. Запустите файл `start.bat` двойным кликом.
4. Откройте в браузере: `http://localhost:3000`.

## Как добавить фото товара

1. Войдите под администратором.
2. Откройте профиль.
3. Перейдите в раздел «Склад».
4. В форме товара выберите файл в поле «Фото товара».
5. Нажмите «Добавить товар» или «Сохранить».

Фото сохраняется вместе с товаром и отображается на карточке компьютера.
